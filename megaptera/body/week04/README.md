### 배운 점

하지만 그냥 이렇게 하면 이렇게 되는구나만 알고 사용했던 것들의 프로세스에 대해 탐구해보게 되었고,\
저번 주까지 더 이상 공부하지 않아도 되겠다고 생각했던 내용들이 지금은 거대한 학습 내용으로 다가오게 되었습니다.
